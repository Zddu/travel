# travel

> 手机端旅游网开发实战

## 安装依赖
npm install

## 运行项目
npm run dev

## 打包项目
npm run build

## 项目介绍
  模仿手机端旅游网站，实现了首屏页面的搭建、城市搜索、按首字母搜索、本地信息存储、
  景点详情、递归组件、swiper轮播封装
